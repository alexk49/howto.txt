### todo.txt basic commands

add new task: td add "task to add +project @context due:yyyy-mm-dd" +todo.txt +new +task

append to a task td append ITEM# "text to append" +todo.txt +append +add 

archive finished tasks / move tasks marked as complete to done.txt: td archive +todo.txt +archive +finish

list tasks that contain particular term: td list "search term" +todo.txt +list +search
